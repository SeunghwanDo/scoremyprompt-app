FUNCTIONALITY
=============

Landing Page Flow:
1. User selects job role
2. User enters prompt to analyze
3. User clicks "Analyze My Prompt"
4. API validates and processes prompt
5. Redirects to results page

Results Page Flow:
1. Displays overall PROMPT Score (0-100)
2. Shows 6 dimension scores with feedback
3. Displays strengths and improvements
4. Compares against benchmarks
5. Offers share functionality
6. CTA to join PromptTribe
7. Option to analyze another prompt

PROMPT Score Dimensions
=======================

P - Precision       [Clarity and specificity]
R - Relevance       [How well-targeted]
O - Organization    [Structure and flow]
M - Methodology     [Clear approach]
P - Personalization [Tailored to context]
T - Timeframe       [Timeline and deliverables]

Each scored 0-100, overall score is average.

SETUP STEPS
===========

1. npm install                  [Install dependencies]
2. cp .env.example .env.local   [Create environment file]
3. Add ANTHROPIC_API_KEY        [Set your API key]
4. npm run dev                  [Start dev server]
5. Open http://localhost:3000   [Access application]

BUILD COMMANDS
==============

Development:  npm run dev       [Start dev server]
Build:        npm run build     [Build for production]
Start:        npm start         [Start production server]
Lint:         npm run lint      [Run ESLint]
