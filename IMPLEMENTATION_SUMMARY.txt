1. /app/api/og/route.jsx (390 lines)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   
   Purpose: Dynamically generate 1200x630px PNG share cards
   Technology: Next.js API Route + @vercel/og (ImageResponse)
   
   Exports:
   - GET(request) - Async handler that accepts query parameters
   
   Features:
   - Grade color mapping (S→Green, A→Blue, B→Amber, C→Orange, D→Red)
   - DimensionBar component for SVG dimension visualization
   - Full parameter validation and score clamping (0-100)
   - Dark gradient background with accent gradients
   - 1200x630px PNG output for optimal social media display
   - Error handling and console logging
   
   Query Parameters Accepted:
   • score (0-100) - Overall score
   • grade (S/A/B/C/D) - Letter grade
   • gradeLabel (text) - Grade description
   • jobRole (text) - Profession/category
   • percentile (0-100) - Percentile ranking
   • p, r, o, m, s, t (0-100 each) - Dimension scores
   
   Response:
   • Content-Type: image/png
   • Dimensions: 1200x630px
   • Caching: Can be cached by CDN
