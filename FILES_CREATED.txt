PROMPTLAB SHARE CARD GENERATOR - FILES CREATED
=============================================================================

Two main files have been created to enable social media sharing with rich preview cards:

1. MAIN API ROUTE: /app/api/og/route.jsx
   - Next.js API route for dynamic OG image generation
   - Uses @vercel/og (ImageResponse) to generate 1200x630 PNG images
   - Accepts query parameters for score, grade, job role, percentile, dimensions
   - Returns a premium dark-themed share card image

2. SHARE PAGE: /app/share/[id]/page.js
   - Dynamic Next.js page for social media sharing
   - Generates server-side OpenGraph and Twitter meta tags
   - Accepts URL parameters and passes them to the OG image API
   - Properly formatted for LinkedIn, X/Twitter, and Facebook sharing

=============================================================================
QUERY PARAMETERS
=============================================================================

All parameters passed to /share/[id]?... are forwarded to /api/og for image generation:

  score         (0-100)    - Overall PROMPT Score
  grade         (S/A/B/C/D) - Letter grade
  gradeLabel    (text)     - Grade description (e.g., "Skilled Prompter")
  jobRole       (text)     - Job role/industry category
  percentile    (0-100)    - Percentile ranking among peers
  p             (0-100)    - Precision dimension score
  r             (0-100)    - Relevance dimension score
  o             (0-100)    - Organization dimension score
  m             (0-100)    - Methodology dimension score
  s             (0-100)    - Personalization dimension score
  t             (0-100)    - Timeframe dimension score

Example URL:
  /share/abc123?score=78&grade=A&gradeLabel=Skilled+Prompter&jobRole=Marketing&percentile=23&p=80&r=75&o=80&m=78&s=70&t=75

Grade Color Mapping:
  S (Exceptional)  → Green  (#10b981) - Score 85+
  A (Skilled)      → Blue   (#3b82f6) - Score 75-84
  B (Competent)    → Amber  (#f59e0b) - Score 60-74
  C (Developing)   → Orange (#f97316) - Score 45-59
  D (Needs Work)   → Red    (#ef4444) - Score <45

Dimension Colors (dynamic based on individual score):
  85+  → Green  (#10b981)
  75-84 → Blue  (#3b82f6)
  60-74 → Amber (#f59e0b)
  <60   → Red   (#ef4444)
