SETUP INSTRUCTIONS
================================================================================

Quick Setup (5 steps):
  1. npm install
  2. cp .env.example .env.local
  3. Add ANTHROPIC_API_KEY to .env.local
  4. npm run dev
  5. Open http://localhost:3000

Development Commands:
  npm run dev      - Start development server
  npm run build    - Build for production
  npm start        - Start production server
  npm run lint     - Run ESLint

Environment Variables:
  ANTHROPIC_API_KEY    - Your Anthropic API key (required)
  NEXT_PUBLIC_API_URL  - API base URL (optional)

To Integrate Claude API:
  1. Install @anthropic-ai/sdk: npm install @anthropic-ai/sdk
  2. Update app/api/analyze/route.js
  3. Replace mock response with actual Claude API call
  4. Use included PROMPT_SCORE_SYSTEM constant
  5. Test with real prompts

The PROMPT Score System prompt is included in app/api/analyze/route.js
and defines the 6-dimension evaluation framework.
